# Typing Blitz

Test and improve your typing speed with this fast-paced typing challenge game.

## Features
- Multiple difficulty modes
- Speed tracking
- Accuracy scoring
- Daily challenges
- Leaderboards

## Build
```bash
npm install
npx cap sync
npx cap open android
npx cap open ios
```

## Tech Stack
- React + Vite
- Capacitor
- iOS & Android
